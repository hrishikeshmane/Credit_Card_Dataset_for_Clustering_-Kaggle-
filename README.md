# Credit_Card_Dataset_for_Clustering_-Kaggle-

The Python Notebook is for [Credit Card Dataset for Clustering](https://www.kaggle.com/arjunbhasin2013/ccdata) problem on [Kaggle](https://www.kaggle.com)

The problem is an example of Unsupervised learning.

This case requires to develop a customer segmentation to define marketing strategy. The sample Dataset summarizes the usage behavior of about 9000 active credit card holders during the last 6 months. The file is at a customer level with 18 behavioral variables.

The solution deals with the problem by segmenting the customers into 8 types by using K-means algorithm

This is also available as a public Kernel on Kaggle- https://www.kaggle.com/ihrishimane26/k-means-for-unsupervised-learning
